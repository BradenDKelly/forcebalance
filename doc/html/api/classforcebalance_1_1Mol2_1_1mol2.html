Detailed Description

This is to manage one mol2 series of lines on the form:

 CDK2.xray.inh1.1E9H
  34 37 0 0 0
 SMALL
 GASTEIGER
 Energy = 0

       1 C1          5.4790   42.2880   49.5910 C.ar    1  <1>         0.0424
       2 C2          4.4740   42.6430   50.5070 C.ar    1  <1>         0.0447
 @<TRIPOS>BOND
      1     1     2   ar
      2     1     6   ar
