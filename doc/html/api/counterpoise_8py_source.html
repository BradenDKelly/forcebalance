00003 Match an empirical potential to the counterpoise correction for basis set superposition error (BSSE).
00004 
00005 Here we test two different functional forms: a three-parameter
00006 Gaussian repulsive potential and a four-parameter Gaussian which
00007 goes smoothly to an exponential.  The latter can be written in two
00008 different ways - one which gives us control over the exponential,
00009 the switching distance and the Gaussian decay constant, and

00012 the third option is the best although our early tests have
00013 indicated that none of the force fields perform particularly well

00094     def get(self,mvals,AGrad=False,AHess=False):
00095         """Gets the objective function for fitting the counterpoise correction.
00096 
00097         As opposed to AbInitio_GMXX2, which calls an external program,
00098         this script actually computes the empirical interaction given the
00099         force field parameters.
00100 
00101         It loops through the snapshots and atom pairs, and computes pairwise
00102         contributions to an energy term according to hard-coded functional forms.
00103 
00104         One potential issue is that we go through all atom pairs instead of
00105         looking only at atom pairs between different fragments.  This means that
00106         even for two infinitely separated fragments it will predict a finite
00107         CP correction.  While it might be okay to apply such a potential in practice,
00108         there will be some issues for the fitting.  Thus, we assume the last snapshot
00109         to be CP-free and subtract that value of the potential back out.
00110 
00111         Note that forces and parametric derivatives are not implemented.
00112 
00113         @param[in] mvals Mathematical parameter values
00114         @param[in] AGrad Switch to turn on analytic gradient (not implemented)
00115         @param[in] AHess Switch to turn on analytic Hessian (not implemented)
00116         @return Answer Contribution to the objective function
00117         
00118         """
00119 
00120         # Create the force field physical values from the mathematical values
00121         pvals = self.FF.create_pvals(mvals)
00122         cpmm = []
00123         logger.info("CPMM: %s   \r" % self.name)
00124         # Loop through the snapshots
00125         for s in range(self.ns):
00126             xyz = self.xyzs[s] # Harvest the xyz. :)
00127             cpmm_i = 0.0
00128             # Loop through atom pairs
00129             for i in range(self.na):
00130                 for j in range(i+1,self.na):
00131                     # Compute the distance between 2 atoms
00132                     # and the names of the elements involved
00133                     dx = linalg.norm(xyz[i,:]-xyz[j,:])
00134                     ai = self.elem[i]
00135                     aj = self.elem[j]
00136                     aiaj = ai < aj and "%s%s" % (ai,aj) or "%s%s" % (aj,ai)
00137                     pidlist = ['CPGAUSSA','CPGAUSSB','CPGAUSSC']
00138                     if all([k+aiaj in self.FF.map for k in pidlist]):
00139                         # Look up the parameter values.  This might be something like 'CPGAUSSBLiCl'.
00140                         A, B, C = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
00141                         # Actually compute the interaction.
00142                         cpmm_i += A * exp (-B * (dx - C)**2)
00143                     # This is repeated for different interaction types...
00144                     pidlist = ['CPGEXPA', 'CPGEXPB', 'CPGEXPG', 'CPGEXPX']
00145                     if all([k+aiaj in self.FF.map for k in pidlist]):
00146                         A, B, G, X0 = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
00147                         a  = 2*A*(X0-B)
00148                         b  = A*(X0**2-B**2)+G
00149                         if dx < X0:
00150                             cpmm_i += exp(-a*dx+b)
00151                         else:
00152                             cpmm_i += exp(-A*(dx-B)**2+G)
00153                     pidlist = ['CPEXPGA1','CPEXPGB','CPEXPGX0','CPEXPGA2']
00154                     if all([k+aiaj in self.FF.map for k in pidlist]):
00155                         A1, B, X0, A2 = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
00156                         B2 = 2 * A2 * X0 - A1
00157                         G  = B - A2 * X0**2
00158                         if dx < X0:
00159                             cpmm_i += exp(-A1*dx+B)
00160                         else:
00161                             cpmm_i += exp(-A2*dx**2 + B2*dx + G)
00162             cpmm.append(cpmm_i)
00163         cpmm = array(cpmm)
00164         cpmm -= cpmm[-1] # This prevents craziness from happening
00165         # Write the results to a file for plotting
