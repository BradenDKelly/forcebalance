3 Match an empirical potential to the counterpoise correction for basis set superposition error (BSSE).
4 
5 Here we test two different functional forms: a three-parameter
6 Gaussian repulsive potential and a four-parameter Gaussian which
7 goes smoothly to an exponential. The latter can be written in two
8 different ways - one which gives us control over the exponential,
9 the switching distance and the Gaussian decay constant, and
10 another which gives us control over the Gaussian and the switching
11 distance. They are called 'CPGAUSS', 'CPEXPG', and 'CPGEXP'. I think
12 the third option is the best although our early tests have
13 indicated that none of the force fields perform particularly well

92  def get(self,mvals,AGrad=False,AHess=False):
93  """Gets the objective function for fitting the counterpoise correction.
94 
95  As opposed to AbInitio_GMXX2, which calls an external program,
96  this script actually computes the empirical interaction given the
97  force field parameters.
98 
99  It loops through the snapshots and atom pairs, and computes pairwise
100  contributions to an energy term according to hard-coded functional forms.
101 
102  One potential issue is that we go through all atom pairs instead of
103  looking only at atom pairs between different fragments. This means that
104  even for two infinitely separated fragments it will predict a finite
105  CP correction. While it might be okay to apply such a potential in practice,
106  there will be some issues for the fitting. Thus, we assume the last snapshot
107  to be CP-free and subtract that value of the potential back out.
108 
109  Note that forces and parametric derivatives are not implemented.
110 
111  @param[in] mvals Mathematical parameter values
112  @param[in] AGrad Switch to turn on analytic gradient (not implemented)
113  @param[in] AHess Switch to turn on analytic Hessian (not implemented)
114  @return Answer Contribution to the objective function
115 
116  """
117 
118  # Create the force field physical values from the mathematical values
119  pvals = self.FF.create_pvals(mvals)
120  cpmm = []
121  print "CPMM: %s \r" % self.name,
122  # Loop through the snapshots
123  for s in range(self.ns):
124  xyz = self.xyzs[s] # Harvest the xyz. :)
125  cpmm_i = 0.0
126  # Loop through atom pairs
127  for i in range(self.na):
128  for j in range(i+1,self.na):
129  # Compute the distance between 2 atoms
130  # and the names of the elements involved
131  dx = linalg.norm(xyz[i,:]-xyz[j,:])
132  ai = self.elem[i]
133  aj = self.elem[j]
134  aiaj = ai < aj and "%s%s" % (ai,aj) or "%s%s" % (aj,ai)
135  pidlist = ['CPGAUSSA','CPGAUSSB','CPGAUSSC']
136  if all([k+aiaj in self.FF.map for k in pidlist]):
137  # Look up the parameter values. This might be something like 'CPGAUSSBLiCl'.
138  A, B, C = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
139  # Actually compute the interaction.
140  cpmm_i += A * exp (-B * (dx - C)**2)
141  # This is repeated for different interaction types...
142  pidlist = ['CPGEXPA', 'CPGEXPB', 'CPGEXPG', 'CPGEXPX']
143  if all([k+aiaj in self.FF.map for k in pidlist]):
144  A, B, G, X0 = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
145  a = 2*A*(X0-B)
146  b = A*(X0**2-B**2)+G
147  if dx < X0:
148  cpmm_i += exp(-a*dx+b)
149  else:
150  cpmm_i += exp(-A*(dx-B)**2+G)
151  pidlist = ['CPEXPGA1','CPEXPGB','CPEXPGX0','CPEXPGA2']
152  if all([k+aiaj in self.FF.map for k in pidlist]):
153  A1, B, X0, A2 = [self.FF.pvals[self.FF.map[k+aiaj]] for k in pidlist]
154  B2 = 2 * A2 * X0 - A1
155  G = B - A2 * X0**2
156  if dx < X0:

158  else:

160  cpmm.append(cpmm_i)

163  # Write the results to a file for plotting
