00036 #     A highly accurate five-point finite difference stencil
00037 #     for computing derivatives of a function.  It works on both
00038 #     scalar and vector functions (i.e. functions that return arrays).
00039 #     Since the function does four computations, it's costly but
00040 #     recommended if we really need an accurate reference value.
00041 # 
00042 #     The function is evaluated at points -2h, -h, +h and +2h
00043 #     and these values are combined to make the derivative according to:
00044 #     http://www.holoborodko.com/pavel/numerical-methods/numerical-derivative/central-differences/
00045 # 
00046 #     How to use: use fdwrap or something similar to generate
00047 #     a one-variable function from the (usually) much more complicated
00048 #     function that we wish to differentate.  Then pass it to this function.
00049 # 
00050 #     Inputs:
00051 #     f  = The one-variable function f(x) that we're differentiating
00052 #     h  = The finite difference step size, usually a small number
00053 # 
00054 #     Outputs:
00055 #     fp = The finite difference derivative of the function f(x) around x=0.
00056 #     
00057 def f1d5p(f, h):
00058     fm2, fm1, f1, f2 = [f(i*h) for i in [-2, -1, 1, 2]]
00059     fp = (-1*f2+8*f1-8*fm1+1*fm2)/(12*h)
00060     return fp

00080 #     A three-point finite difference stencil.
00081 #     This function does either two computations or three,
00082 #     depending on whether the 'center' value is supplied.
00083 #     This is done in order to avoid recomputing the center
00084 #     value many times.
00085 # 
00086 #     The first derivative is evaluated using central difference.
00087 #     One advantage of using central difference (as opposed to forward
00088 #     difference) is that we get zero at the bottom of a parabola.
00089 # 
00090 #     Using this formula we also get an approximate second derivative, which
00091 #     can then be inserted into the diagonal of the Hessian.  This is very
00092 #     useful for optimizations like BFGS where the diagonal determines
00093 #     how far we step in the parameter space.
00094 # 
00095 #     How to use: use fdwrap or something similar to generate
00096 #     a one-variable function from the (usually) much more complicated
00097 #     function that we wish to differentate.  Then pass it to this function.
00098 # 
00099 #     Inputs:
00100 #     f  = The one-variable function f(x) that we're differentiating
00101 #     h  = The finite difference step size, usually a small number
00102 # 
00103 #     Outputs:
00104 #     fp = The finite difference derivative of the function f(x) around x=0.
00105 #     
00106 def f12d3p(f, h, f0 = None):
00107     if f0 == None:
00108         fm1, f0, f1 = [f(i*h) for i in [-1, 0, 1]]
00109     else:
00110         fm1, f1 = [f(i*h) for i in [-1, 1]]
00111     fp = (f1-fm1)/(2*h)
00112     fpp = (fm1-2*f0+f1)/(h*h)
00113     return fp, fpp
